# CS-scholars-backend

Backend for CS564 project in UW-Madison

## quick start

make sure that you have configured [Java](https://www.java.com/en/) and [Maven](https://maven.apache.org/) environment.

1. compile the project

```bash
mvn compile
```


2. run the project

```bash
mvn exec:java
```

3. run tests

```bash
mvn test
```
